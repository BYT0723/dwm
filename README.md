# Dwm

## Preview

![preview](./images/dwm-preview.png)

## Install

```shell
git clone https://github.com/BYT0723/dwm.git

cd dwm
chmod +x install.sh

./install.sh

```

## Required

- picom
- xautolock
- mate-power-manager
- nm-applet
- volumeicon
- fcitx5
- udiskie
- trojan
