## Install

```shell
git clone https://github.com/BYT0723/dwm.git

cd dwm
chmod +x install.sh

./install.sh

```

## Required

- picom
- xautolock
- nm-applet
- fcitx5
- udiskie
- imlib (for status bar tab icon)

  Archlinux:

  ```shell
  sudo pacman -S imlib2
  ```

  Debian:

  ```shell
  sudo apt install libimlib2-dev
  ```
